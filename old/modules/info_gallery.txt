Стандартный шаблон ссылается на папку uploads, или папку для медиаданных, установленную по-умолчанию
Если шаблон загружается с параметром default, то он ссылается на папку с названием модуля, т.е. 'gallery'
Если же параметр шаблона задан, то он ссылкается на подпапку с названием параметра внутри папки 'gallery'

Если в этой папке он обнаруживает файл 'captions.ini', то берет заголовки отуда.
Это должен быть файл в формате json, который имеет вид:

{
	"image1.jpg" : "", // заголовок для файла image1.jpg
	"image2.jpg" : ""
}

В нем также разрешена поддержка языков:

{
	"image1.jpg" : {
		"ru" : "", // русский заголовок для файла image1.jpg
		"en" : "" // английский заголовок для файла image1.jpg
	},
	"image2.jpg" : ""
}

Также заголовки могут быть созданы индивидуально для каждого файла.
Тогда они должны иметь то же название, что и оригинальный файл, но расширение "ini".
Файл в формате json, имеет вид:

"заголовок"

или с поддержкой языков:

{
	"ru" : "", // русский заголовок
	"en" : "" // английский заголовок
}

Все спецсимволы в заголовке преобразуются в кодовые знаки для обеспечения безопасности

Классы
======

Классы стандартного шаблона:

name__type и name_param__type

type:
	item - элемент (div)
		link - ссылка (a)
			image - картинка внутри ссылки (img)
		caption - заголовок или описание (div)

Библиотеки
==========

Модуль использует библиотеку fancybox
Проверьте, что она включена в настройках шаблона!
