Модуль вывода новостей и статей из соцсетей

	сканирует заданную стену в соцсети
	читает оттуда информацию (в формате json)
	выводит в шаблон
	
Шаблон задается самостоятельно, после вызова модуля, работа в шаблоне идет с объектом $article (примеры шаблонов собраны в папке templates)

{
	"api" : "vk",
	// значения: string: vk/instagram
	
	"id" : "-77967393",
	// значения: num, string
	// для vk аккаунтов число
	// для vk групп число с минусом впереди
	// для instagram нужно получить при регистрации приложения
	
	"count" : "4",
	// значения: num, string
	// число выводимых новостей
	
	"key" : "abcdefghijklmnopqrstuvwxyzabcdefghijklmnopqrstuvwxyz",
	// значения: string
	// ключ доступа или токен
	
	"disable" : ["235"],
	// значения: array num/string
	// id-номера постов, которые не нужно отображать
	
	"rules" : "text",
	// значения: string
	// правила отображения/блокировки (для инстаграма не действуют)
		// text - отображать только с текстом,
		// images - только с картинками,
		// both - оба
	
	"defaults" : {}
	// значения: array string
	// правила для задания умолчаний (с инстаграмом сейчас не работают)
		// text - если в посте не будет текста
		// images - если в посте не будет картинки
		// можно задавать также с разделением по языку, например text : { ru : '', en : '' }
	
}
