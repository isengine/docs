Модуль чтения таблицы в формате csv и вывода ее в виде массива данных

	читает заданный файл таблицы в формате csv
	читает, если есть, файл настроек (с таким же именем, как у таблицы, но с расширением ini)
	если файла настроен нет, берет значения по-умолчанию
	построчно обрабатывает файл таблицы и записывает в виде массива
	выводит таблицу в шаблон, с правилами объединения ячеек
	
Шаблон задается самостоятельно, после вызова модуля, работа в шаблоне идет с объектом $article (примеры шаблонов собраны в папке templates)

Настройки задаются в файле с расширением .ini и таким же названием, как таблица:
	row -  число колонок, если не указано, будет вычислено самостоятельно
	header - число строк, которые являются заголовком таблицы, если не указано, то считается первая строка
	titlerow - колонка, которая содержит названия подразделов, если не указано, то считается вторая колонка
	name - название или идентификатор таблицы, используется для создания класса, если не указано, то ставится 'default'
	keys - в виде массива ["", "" ...], содержащего названия колонок, если не задан, то идет циферная нумерация, начиная с нуля,
	maxcols - число выводимых строк (0 - все)
	alias - задает номер столбца, содержащего алиас, который в обработчике будет подставлен в качестве идентификатора новости (например, item=mynew вместо item=0)
