# Сравнение данных

Пространство имен:

    use is/Helpers/Match;

## Match::equal

    Match::equal($haystack, $needle)

Функция сравнивает данные, приводя их к строке.

**haystack**
Тип: **смешанный**
Исходные данные

**needle**
Тип: **смешанный**
Сравниваемые данные

Возвращает **true** при совпадении и **null** в противном случае.

## Match::numeric

    Match::numeric($haystack, $min = null, $max = null)
	
Функция сравнивает число в заданном диапазоне.

**haystack**
Тип: **число**
Исходные данные

**min**
Тип: **число**
Минимальное значение

**max**
Тип: **число**
Максимальное значение

Функция сравнивает число в диапазоне между минимальным и максимальным значениями включительно.

Если минимальное значение не задано, то оно считаются в минус бесконечность. Если не задано максимальное значение, то оно считаются в плюс бесконечность.

Возвращает **true** при совпадении и **null** в противном случае.	

## Match::equalIn

    Match::equalIn($haystack, $needle, $and = true)
	
Функция задействует сравнение с массивом значений, приводя данные к строке

**haystack**
Тип: **массив**
Исходные данные

**needle**
Тип: **смешанный**
Сравниваемые данные

**and**
Тип: **смешанный**
Тип сравнения

Сравнение идет с перебором значений в массиве **haystack**. Аргумент **and** управляет типом сравнения:

* по логическому **"и"**, если включено (совпадение всех значений)
* по логическому **"или"**, если выключено (совпадение хотя бы одного значения)

Возвращает **true** при совпадении и **null** в противном случае.

## Match::equalOf

    Match::equalOf($haystack, $needle, $and = true)

Функция задействует сравнение, приводя данные к строке. Cравнение идет по массиву needle.

Фактически является синонимом функции **Match::equalIn** с измененным порядком аргументов. Создана исключительно для поддержки логики в разработке.

Если минимальное значение не задано, то оно считаются в минус бесконечность. Если не задано максимальное значение, то оно считаются в плюс бесконечность.

* по логическому **"и"**, если включено (совпадение всех значений)
* по логическому **"или"**, если выключено (совпадение хотя бы одного значения)
