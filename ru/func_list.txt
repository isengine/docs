Список функций

string
	работа со строками
	
	match
		Функция проверки наличия подстроки или символа в заданной строке
	find
		Функция поиска подстроки или символа в заданной строке
	get
		Функция возвращает подстроку по указанному индексу (позиции) и заданной длины
	cut
		Функция удаления части строки с начала или с конца (при отрицательном значении)
	add
		Функция добавления значений в начало или в конец строки
	reverse
		Функция разворота строки задом наперед
	first
		Функция возврата первого символа строки
	last
		Функция возврата последнего символа строки
	len
		Функция возврата длины строки
	split
		Функция разбивает строку на массив данных по указанным символам
	+ replace
		замена

+regexp
	match
	replace

system
	базовые системные функции
	
	print
		выводит на экран
	console
		выводит в консоль
	set
		определить значение переменной
	type
		определить тип переменной

data
	работа с данными
	
	Данными в общем представлении является ассоциированный массив в php, в js, т.к. ассоциированных массивов нет, используется объект, однако в случае с неассоциированным массивом, ключам присваиваются числовые индексы
	
	is
		функция проверяет, является ли переменная системными данными
	convert
		функция преобразует любые входные данные в системный объект
	get
		Функция возвращает срез массива по указанному индексу (позиции) и заданной длине
	len
		Функция возврата длины массива
	each
		перебор массива
	parse
		Функция парсинга системных данных в системный объект
	match (-)
		Функция проверки на совпадение элемента/ов needle в haystack
	+ sort by key/item abc/zyx natural/random
	cut
	add(item, key:val/val to new numeric key from last numeric key)
	reverse
	first(item, key:key,val:val/key/val)
	last(item, key:key,val:val/key/val)
	+ keys - возврат всех ключей
	+ values - возврат всех значений
	+ combine/rename - назначает новые ключи значениям массива согласно переданному массиву ключей
	+ чистит массив от пустых элементов
	+ чистит массив от одинаковых значений, оставляя первое
	+ find/match поиск ключа/значения в массиве (возможно, через preg_grep)
	+ replace - заменить элементы и ключи массива новыми элементами
	+ склеить значения массива обратно в строку с определенными правилами, разделителями и т.п.
